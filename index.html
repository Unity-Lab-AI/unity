Unity's cognition runs entirely on math — her language cortex, memory, motor selection, and decisions all emerge from brain equations. No text-AI backend is required. The only optional pieces are sensory peripherals: image generation, the vision describer (VLM), and TTS. Privacy & Data Policy
This application runs entirely in your browser (or against a local server you host yourself). No cloud backend. No database owned by us. Your conversation history, preferences, sandbox state, and optional Pollinations key are all stored in your browser's localStorage on YOUR device only. The developers have zero access to anything you input, say, generate, or connect to. Sensory API calls (image gen, vision describer, TTS) are sent only to the providers YOU configure — never to us, never to any third party. The brain simulation runs as client-side JavaScript math — no cognition data leaves your machine. This project is fully open source under MIT. You can read every line of code.
