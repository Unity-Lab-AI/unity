Privacy & Data Policy
This application runs entirely in your browser. No server. No backend. No database. Your API keys, conversation history, preferences, and all data are stored in your browser's localStorage on YOUR device only. The developers have zero access to anything you input, say, generate, or connect to. API keys are sent only to the AI provider YOU choose (Pollinations, OpenAI, etc.) — never to us, never to any third party. The brain simulation runs as client-side JavaScript math — no data leaves your machine except your direct API calls. This project is fully open source. You can read every line of code.
