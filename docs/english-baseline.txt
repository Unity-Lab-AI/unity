# English Linguistic Baseline Corpus
#
# This file is NOT Unity's personality. It's the linguistic competence
# any 25-year-old American English speaker would have — the vocabulary
# and conversational patterns learned from growing up speaking English.
#
# Unity loads this alongside her persona file (Ultimate Unity.txt).
# The persona gives her WHO she is; this file gives her the English
# she uses to express it. Neutral register, first-person casual American.
#
# No goth/emo/drugs/coding themes — that content belongs in the persona.
# This is the base layer: how to say hi, how to ask questions, how to
# describe a day, how to react, how to agree or disagree, how to talk
# about what you did or what you want.
